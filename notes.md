# Day 01 - HTML Fundamentals

## What I learned
- Semantic HTML means using tags that describe their content
- header, nav, main, article, section, aside, footer are semantic tags
- div and span are non-semantic (no meaning by themselves)
- <time> tag is for dates
- <strong> is for important text, <em> is for emphasis
- <ul> is unordered list, <ol> is ordered list
- Two pages link to each other using href="pagename.html"
