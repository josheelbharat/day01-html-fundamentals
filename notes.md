# Day 01 - HTML Fundamentals

## What I learned
- Semantic HTML means using tags that describe their content
- header, nav, main, article, section, aside, footer are semantic tags
- div and span are non-semantic (they mean nothing by themselves)
- The <time> tag is for dates, <strong> for important text, <em> for emphasis
- Two pages can link to each other using href="pagename.html"

- about.html — about page linked from the nav
